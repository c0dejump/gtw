# GWT


Github to Wordlist     
Just a little tool to turn a github repository into a wordlist


# Usage
  
```  
  usage: gwt.py [-h] [-u GIT_URL]
  
  optional arguments:
    -h, --help  show this help message and exit
    -u GIT_URL  URL git [required]. Ex: '-u https://github.com/c0dejump/HawkScan.git'
```
